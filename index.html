Steps to set up your dev environment:

  • Install Visual Studio Code: https://code.visualstudio.com/
  • Install Git: https://git-scm.com/
  • Set up your git username: git config --global user.name "Your Name"
  • Set up your git email: git config --global user.email "youremail@example.com"
  • Generate a key pair to allow you to authenticate to GitHub: ssh-keygen -t rsa -b 4096. On Windows, you may need to use `git bash` to use that command.
  • Add your new public key to GitHub via their control panel.
  • Install GitHub Extension: In VS Code, install the GitHub Pull Requests and Issues extension.
  • Open up your 'console' or 'terminal'.
  • Inside your terminal, navigate to a directory where you want to put your project folder. (use the 'cd' command)
  • Clone or create repository into the current folder using the command: clone git@github.com:jondubois/ga-site.git (example)
  • Navigate to your new project folder using the `cd` command.
  • Open that folder in Visual Studio Code e.g. use the code . command to open the current folder from the console/terminal.
  • Create a new .html file inside Visual Studio Code and write some HTML in it.
  • git add .
  • git commit -m "This my first commit message"
  • git push origin main
